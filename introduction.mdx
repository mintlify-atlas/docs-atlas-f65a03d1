# Orange Juice makes Hacker News sweeter

Orange Juice is a browser extension that enhances your Hacker News experience with small, thoughtful improvements. Built with WXT and React, it adds features and polish that make HN more pleasant to use without changing its core interface.

## What is Orange Juice?

Orange Juice is a browser extension for Chrome and Firefox that enhances Hacker News with quality-of-life improvements. The extension focuses on subtle enhancements that feel natural and aligned with HN's minimal design philosophy.

The goal isn't to redesign Hacker News—its minimal UX works, and no one wants to relearn an interface that already does the job. Instead, Orange Juice adds features and polish that make HN more pleasant to use without changing its core.

<Info>
  Orange Juice has been developed by a long-time HN community member (since 2009) who accumulated ideas over the years for improving the experience.
</Info>

## Key features

Orange Juice provides dozens of enhancements that should feel invisible but improve your daily HN browsing:

### Comment enhancements

- **Inline reply** - Reply to comments directly on the page without navigating away, with support for quoting selected text
- **Highlight unread comments** - New comments are highlighted with an orange border when you revisit threads within 3 days
- **Mermaid diagram rendering** - Automatically renders diagrams wrapped in `<mermaid>` blocks
- **Improved code formatting** - Better styling for code blocks and inline code snippets
- **Clickable indent to collapse** - Click comment indentation to collapse/expand threads
- **OP highlighting** - Original poster's username marked with `[op]` label in orange
- **Auto-resize textarea** - Comment textareas automatically adjust height based on content
- **Collapse root button** - Collapse all top-level comments at once on thread pages

### Story enhancements

- **Hide read stories** - Checkbox at the top of story pages to hide already-visited links using browser history
- **Keyboard navigation** - Navigate stories and comments efficiently with keyboard shortcuts
- **Favorite items** - Save comments and stories for later reference with an inline favorite button

### User experience

- **Dark mode** - Site-wide dark mode that preserves HN's visual structure while improving readability
- **User info on hover** - Hover over usernames to see account details, karma, and about section
- **Profile links dropdown** - Quick access to your profile, submissions, comments, and favorites from the top nav
- **Enhanced submission UX** - Character counter for titles and "fetch title" button for automatic title retrieval

### Navigation

- **Past page date picker** - Navigate historical front pages with an interactive date picker on `/front`
- **More links dropdown** - Access hidden HN URLs that aren't exposed in the regular interface
- **Profile about clickable links** - Links in user about sections become clickable

<Tip>
  Ideally, you shouldn't really notice Orange Juice at all. The features feel subtle but natural: visible, useful, and aligned with what most users would expect the site to have supported in the first place.
</Tip>

## Built with modern tools

Orange Juice is built using:

- **WXT** - Modern web extension framework for Chrome and Firefox
- **React** - Component-based UI library
- **TypeScript** - Type-safe JavaScript development
- **Tailwind CSS** - Utility-first CSS framework
- **Bun** - Fast JavaScript runtime and package manager
- **Vitest** - Unit testing framework

## Open source and community-driven

Orange Juice is fully open source and available on GitHub. Keeping the extension open source is critical for trust—the last thing the project aims to be is a liability to the HN community.
