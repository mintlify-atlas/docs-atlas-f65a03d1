## Overview

Orange Juice includes a comprehensive set of utilities and services that power the extension's features.

## Storage Services

### Local Storage

**Location:** `src/utils/local-storage.ts`

Wrapper around `@wxt-dev/storage` for persistent data storage.

```typescript
import { lStorage } from '@/utils/local-storage';

// Get item
const value = await lStorage.getItem<string>('key');

// Set item
await lStorage.setItem('key', 'value');

// Clear all storage
await lStorage.clear();
```

### Activity Trail

**Location:** `src/utils/activity-trail.ts`

Manages user activities (favorites, flags, votes, hidden items) with indexed storage for fast lookups.

<ParamField path="ActivityTrail" type="class">
  Tracks and indexes all user activities for quick lookups
</ParamField>

```typescript
import { newActivityTrail } from '@/utils/activity-trail';

const activityTrail = await newActivityTrail();

// Check if item is favorited
const isFavorited = activityTrail.get({
  id: 'item123',
  type: ActivityId.Favorites
});

// Add activity
activityTrail.set({
  id: 'item123',
  type: ActivityId.Favorites,
  exp: Date.now() + 86400000
});
```

### Activity Fetcher

**Location:** `src/utils/activity-trail.ts`

Fetches all user activities from Hacker News and syncs to local storage.

```typescript
import { newActivityFetcher } from '@/utils/activity-trail';

const fetcher = newActivityFetcher('username');
await fetcher.start(); // Syncs all activities
```

## DOM Utilities

**Location:** `src/utils/dom.ts`

### getUsername

Extracts and caches the current username from the page.

```typescript
import { getUsername } from '@/utils/dom';

const username = getUsername(document);
```

### getAuthToken

Fetches authentication token for activities (favorite, flag, vote).

```typescript
import { getAuthToken } from '@/utils/dom';

const token = await getAuthToken('comment123', 'favorite');
```

### getAllComments

Retrieves all comment elements from the page.

```typescript
import { getAllComments } from '@/utils/dom';

const comments = getAllComments(document);
```

### getPageDom

Fetches and parses a remote HN page.

```typescript
import { getPageDom } from '@/utils/dom';

const remoteDom = await getPageDom('https://news.ycombinator.com/item?id=123');
```

## Dark Mode

**Location:** `src/utils/dark-mode.ts`

Manages dark mode preference and applies theme changes.

### Functions

```typescript
import {
  getDarkModePreference,
  setDarkModePreference,
  toggleDarkModePreference,
  enableDarkMode
} from '@/utils/dark-mode';

// Get current preference
const mode = await getDarkModePreference(); // 'dark' | 'light'

// Set preference
await setDarkModePreference('dark');

// Toggle
await toggleDarkModePreference();

// Initialize dark mode
enableDarkMode(document);
```

### CSS Class

Dark mode adds `oj-dark-mode` class to the `<html>` element.

## Indexed List

**Location:** `src/utils/indexed-list.ts`

Efficient list navigation with O(1) key-based lookups.

```typescript
import { IndexedList } from '@/utils/indexed-list';

const list = new IndexedList(items, (item) => item.id);

// O(1) lookup
const item = list.get('key123');

// Navigate
const next = list.getNext(currentItem);
const prev = list.getPrevious(currentItem);

// Boundaries
const first = list.first();
const last = list.last();
```

import {

## Reference Link Parser

**Location:** `src/utils/parse-reference-links.ts`

Extracts numbered reference links from comments.

```typescript
import { parseReferenceLinks } from '@/utils/parse-reference-links';

const links = parseReferenceLinks(commentElement);
// Returns: [{ href: 'https://...', index: 1 }, ...]
```

Matches patterns like:
- `[1]`, `[2]`, etc.
- `1:`, `2:`, etc.
- `1.`, `2.`, etc.

## Backoff Utility

**Location:** `src/utils/backoff.ts`

Retries failed operations with exponential backoff.

```typescript
import { withBackoff } from '@/utils/backoff';

const result = await withBackoff(
  async () => {
    return await fetchData();
  },
  {
    attempts: 3,
    initialDelay: 1000
  }
);
```
