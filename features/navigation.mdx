## Keyboard Shortcuts

Press `?` on any page to open the keyboard shortcuts help dialog.

## Past Page Date Picker

Navigate historical front pages with an interactive date picker on the `/front` page.

### Features

- **Date range**: Select any date from 2007 to yesterday
- **Smart defaults**: Defaults to yesterday's front page
- **Dynamic validation**: Automatically prevents selecting future dates
- **Adaptive UI**: Available months and days adjust based on today's date

### How It Works

<Steps>
  <Step title="Year selection">
    Choose a year from 2007 to the current year. The dropdown is populated in descending order.
  </Step>
  
  <Step title="Month selection">
    Available months adjust if you select the current year (can't select future months).
  </Step>
  
  <Step title="Day selection">
    Available days adjust if you select the current year and month (can't select future dates or tomorrow).
  </Step>
  
  <Step title="Navigation">
    Changing any dropdown immediately navigates to `/front?day=YYYY-M-D` with the selected date.
  </Step>
</Steps>

### Implementation Details

```typescript
// src/components/past/index.ts:48-49
const today = new Date();
today.setDate(today.getDate() - 1); // Default to the day before
```

The date picker uses `updateMonthAndDayOptions()` to dynamically update available options:

```typescript
const isCurrentYear = newYearValue === today.getFullYear();
const maxMonth = isCurrentYear ? today.getMonth() + 1 : 12;

const isCurrentMonth = isCurrentYear && newMonthValue === today.getMonth() + 1;
const maxDay = isCurrentMonth ? today.getDate() + 1 : 31;
```

<Warning>
  The past page date picker only appears on `/front` pages and requires login.
</Warning>
